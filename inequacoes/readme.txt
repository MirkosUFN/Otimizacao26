Construir um algoritmo que trace a região no primeiro quadrante, para a solução de um sistema de inequações.
O software deve pedir ao usuário as inequações na forma de a1X + a2Y SINAL valorDireito, onde SINAL é um dos valores
da lista [<, <=, >, >=] e valorDireito é um valor numérico inteiro ou fracionário.
Quando o algoritmo recebe mais de uma inequação, a região destacada no gráfico é aquela que representa a intersecção
entre as áreas de cada inequação.

O nome do arquivo deve ser ineq-[nomeAluno].extensão onde extensão representa o tipo do arquivo da linguagem escolhida 
(de livre escolha).
